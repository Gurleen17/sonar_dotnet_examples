Tools
=======
Nunit - is an open source unit testing framework for Microsoft .NET
OpenCover - is a code coverage tool for .NET 2 and above, support for 32 and 64 processes with both branch and sequence points
ReportGenerator - converts XML reports generated by PartCover, OpenCover or NCover into a readable report in various formats
OpenCoverToCoberturaConverter - converts OpenCover reports to Cobertura reports
